# BDAI - Gestor de Música

Aplicação web para gestão de uma biblioteca musical (Artistas, Álbuns, Músicas e Playlists), inspirada em serviços de streaming.
Desenvolvido no âmbito da disciplina de Bases de Dados e Análise de Informação (BDAI), lecionada pelo Professor Pedro Furtado na Universidade de Coimbra no ano académico de 2025/2026.

### Sobre o Projeto
Este projeto demonstra a implementação de um modelo de base de dados relacional utilizando **Django** e **PostgreSQL**. O foco principal é a correta modelação de dados e a sua interação via aplicação web.

### Tecnologias Usadas

**Backend:**
* Python 3
* Django 5
* PostgreSQL

**Frontend:**
* HTML5 / CSS3
* Bootstrap 5 (Framework CSS)
* Django Template Language (DTL)
* django-bootstrap-v5

**Ferramentas:**
* VS Code
* pgAdmin 4
